- Pastas: `diaNN-tema-curto/` (ex.: `dia04-art5-cf88`).
- PDF final profissional: `..._PRO.pdf` (gerado de HTML via Chrome).
- Provas no layout fiel da banca: geradas por `engine/gerar_prova_quadrix.py`.
